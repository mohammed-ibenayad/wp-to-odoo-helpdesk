How to Get Your API Key:

  1. Log into your Odoo instance
  2. Go to Preferences (your profile)
  3. Click Account Security tab
  4. Click New API Key
  5. Enter description: "WhatsApp Integration"
  6. Copy the generated key and paste above

How to Use:

  1. Configure your connection above and test it
  2. Go to WhatsApp Web in a new tab
  3. Open any conversation
  4. Single Message: Hover over a message → Click action buttons
  5. Multi-Select: Click "Multi-Select" → Select messages → Create item
  6. Choose custom titles and check your Odoo for results!
