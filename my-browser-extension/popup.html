How to Get Your API Key:

  1. Log into your Odoo instance
  2. Go to Preferences (your profile)
  3. Click Account Security tab
  4. Click New API Key
  5. Enter description: "WhatsApp Integration"
  6. Copy the generated key and paste above

How to Use:

  1. Enter your Odoo URL and username
  2. Get your API key from Odoo (see above)
  3. Test the connection
  4. Go to WhatsApp Web in a new tab
